Complex Return Checklist

Declaración Compleja

For multi-state filing, rental properties, investments, or IRS issues.
Múltiples estados, rentas, inversiones o problemas con el IRS.

Core Documents Requerido

  • Photo ID & SSN Cards
  • Prior Year Return / Declaración 2023 Essential for tracking depreciation and loss carryovers.

Complexity Items Detalles

  • Multi-State Residency / Múltiples Estados Dates you lived/worked in each state during 2024.
  • Rental Properties / Rentas Income received and expense breakdown (repairs, insurance, taxes) per property.
  • Investments / Inversiones 1099-B forms for stocks and detailed crypto reports (gain/loss).
  • IRS Notices / Cartas del IRS Any recent letters (CP2000, etc.) from IRS or State.
  • Prior Year Gaps Explanation if you missed filing any previous years.
