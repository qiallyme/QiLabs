## KDP (KINDLE DIRECT PUBLISHING) CHECKLIST

### Pre-Publication

- [ ] **Manuscript Ready**
  - [ ] All 28 files processed and validated
  - [ ] MASTER_BOOK.md created
  - [ ] Exported to EPUB format (for Kindle)
  - [ ] Exported to PDF format (for paperback)
  - [ ] Tested on Kindle Previewer
  - [ ] Table of contents verified
  - [ ] Page breaks correct
  - [ ] No formatting errors

- [ ] **Cover Design**
  - [ ] Front cover (eBook: 2560x1600px minimum)
  - [ ] Full wrap cover (Paperback: includes spine and back)
  - [ ] Spine width calculated based on page count
  - [ ] Back cover copy written
  - [ ] Cover meets KDP requirements (bleed, margins, etc.)

- [ ] **Metadata**
  - [ ] Title and subtitle finalized
  - [ ] Author name consistent
  - [ ] Description written (long and short)
  - [ ] BISAC categories selected
  - [ ] Keywords researched and selected
  - [ ] Publication date set

- [ ] **ISBN**
  - [ ] Decide: Use KDP free ISBN or purchase your own
  - [ ] If purchasing: Buy from Bowker (US) or Nielsen (UK)
  - [ ] ISBN assigned to manuscript

- [ ] **Pricing**
  - [ ] eBook price set
  - [ ] Paperback price set (based on page count)
  - [ ] Royalty structure understood (35% vs 70% for eBook)
  - [ ] Kindle Unlimited enrollment decision made

- [ ] **Territories**
  - [ ] Worldwide rights confirmed
  - [ ] Or specific territories selected

### Upload Process

- [ ] **eBook Upload**
  - [ ] EPUB file uploaded
  - [ ] Cover image uploaded
  - [ ] Preview checked on all device types
  - [ ] Metadata entered
  - [ ] Pricing set
  - [ ] Preview approved

- [ ] **Paperback Upload**
  - [ ] PDF manuscript uploaded
  - [ ] Full wrap cover uploaded

  - [ ] Proof reviewed and approved
  - [ ] Final approval given

### Post-Publication

- [ ] **Amazon Author Central**
  - [ ] Author profile created/updated
  - [ ] Author photo uploaded
  - [ ] Bio added
  - [ ] Book linked to profile

- [ ] **Marketing**
  - [ ] Amazon product page optimized
  - [ ] A+ Content added (if eligible)
  - [ ] Author page updated
  - [ ] Social media announcements posted

## IMPORTANT NOTES

### ISBN Considerations

**KDP:**
- Free ISBN available (but limits distribution)
- Can use your own ISBN for more control
- If using KDP ISBN, cannot use same ISBN on IngramSpark

**IngramSpark:**
- **MUST use your own ISBN**
- Cannot use KDP-assigned ISBN
- Purchase from Bowker (US) or Nielsen (UK)

### Simultaneous Publishing

You CAN publish on both KDP and IngramSpark, but:
- Use different ISBNs for each platform
- Or use your own ISBN on both (but this can cause distribution conflicts)
- Best practice: Use your own ISBN on IngramSpark, KDP's free ISBN on Amazon

### Pricing Strategy

**KDP:**
- eBook: 35% or 70% royalty (70% requires $2.99-$9.99 price range)
- Paperback: 60% royalty (after printing costs)

**IngramSpark:**
- Set wholesale discount (typically 55%)
- Retailers set final price
- You receive wholesale price minus printing costs

## TIMELINE ESTIMATE

**KDP:**
- eBook: 24-48 hours after approval
- Paperback: 3-5 business days after proof approval

**IngramSpark:**
- Digital proof: 24-48 hours
- Physical proof: 5-7 business days (if ordered)
- Distribution: 1-2 weeks after final approval
