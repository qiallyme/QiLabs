How to Run SmartRename AI on Your ComputerThis application is built using React and Tailwind CSS. To run it, you need to provide a Google Gemini API Key and set up a local development environment.1. Get your Gemini API KeySince this app uses AI to "read" your files, you need a key from Google:Go to Google AI Studio.Click "Get API key".Create a new API key in a new project.Important: In the smart_rename.jsx file, find const API_KEY = ""; and paste your key between the quotes.2. Option A: Quick Running (Development Environment)The easiest way to run this locally is using Vite.Install Node.js: Download it from nodejs.org if you don't have it.Create a project: Open your terminal/command prompt and run:npm create vite@latest smart-rename -- --template react
cd smart-rename
npm install
Install Dependencies:npm install lucide-react
Setup Tailwind CSS:Follow the Tailwind CSS Vite guide to initialize Tailwind in your project.Replace the Code:Open src/App.jsx in your project and replace its entire content with the code from the smart_rename.jsx tab on the right.Run it:npm run dev
Open the URL provided in your terminal (usually http://localhost:5173).3. Important Notes on File AccessBrowser Security: For security reasons, web browsers cannot directly "rename" files on your hard drive.The Workflow: This app works by "Processing" the file in memory and then providing a Download button. When you download it, it saves the file with the new name.Undo: The "Undo" feature in the app reverts the change within the app's memory/log so you can try again before downloading.4. Troubleshooting "Unknowns"If the AI is still giving you "Unknown" values:Ensure the file is a standard PDF, Image, or Text file.Use the Context Hints box. For example, type: "These are medical invoices from 2024. Extract the doctor name as Organization."
