# QiLegacy Estate Questionnaire Field Map

This field inventory was extracted from the legacy packet files in this directory and translated into the new app workflow.

## Packet scope

1. Last Will and Testament
2. Living Will and Health Care Power
3. Durable Power of Attorney (Financial)
4. HIPAA Authorization
5. Revocable Living Trust (Optional)
6. Beneficiary Designations Summary
7. Final Wishes Addendum
8. Signature and storage checklist

## Core identity fields

- Principal full legal name
- Preferred name
- Date of birth
- Address line 1
- Address line 2
- City
- State
- Postal code
- Jurisdiction state
- Family context notes
- Dependents and guardianship notes

## Key people / role assignments

- Executor / personal representative
- Alternate executor
- Primary health care agent
- Alternate health care agent
- Attorney-in-fact
- Successor trustee
- Guardian for minors or dependents
- Primary HIPAA recipient

Each role needs:

- Full legal name
- Relationship
- Mailing address
- Phone
- Email

## Will fields

- Primary beneficiary
- Alternate beneficiary
- Specific bequests and distribution notes
- Guardianship notes
- Tax and final-expense instructions
- Funeral-reference note

## Living will / health care directive fields

- Life-support preference
- CPR / DNR / DNI preference
- Feeding-tube preference
- Comfort-care and pain-management preference
- Organ-donation preference
- Autopsy preference
- Additional medical instructions

## Financial POA fields

- Effective date
- Powers granted
- Benefits authority
- Property authority
- Special limits or exclusions

## HIPAA fields

- Release scope
- Expiration rule
- Additional recipients or restrictions

## Trust fields

- Trust status / whether the trust is being used
- Trust name
- Trust purpose
- Schedule A property notes
- Revocation terms

## Beneficiary and asset fields

### Asset inventory

- Category
- Description
- Location / institution hint
- Intended recipient
- Trust-funding flag
- Notes

### Beneficiary accounts

- Institution name
- Account label
- Account type
- Primary beneficiary
- Contingent beneficiary
- Percentage / split notes
- Reviewed-on date

## Final wishes fields

- Disposition of remains
- Preferred funeral home
- Burial or memorial location
- Memorial music
- Memorial readings / speakers
- Memorial theme
- Personal message
- Digital vault location
- Digital executor notes

## Signature and execution fields

- Witness count
- Notary requirement
- Storage plan
- Attorney review status
- Final execution checklist
